## 📒개발 환경
- `java 8`
- `JDK 1.8.0`
- **IDE** : STS 3.9
- **Framework** : Spring
- **Database** : Oracle DB(11xe)
- **ORM** : Mybatis

## 💡주요 기능
#### 로그인
- DB값 검증
- ID찾기, PW찾기
- 로그인 시 쿠키(Cookie) 및 세션(Session) 생성

#### 회원가입 및 회원 정보 수정, 탈퇴
- 주소 API 연동
- 회원 정보 수정
- 회원 탈퇴

#### 게시판 CRUD
- 게시물 리스트 및 페이징
- 게시물 검색 처리
- 게시물 등록
- 게시글 수정
- 게시글 삭제

#### 댓글 CRUD
- 댓글 등록
- 댓글 수정
- 댓글 삭제
- 댓글 페이징 처리
